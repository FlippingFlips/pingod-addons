# pingod-netproc-pdb

An example running a game with a P-ROC device or a Fake P-ROC device with PinGodGame / NetProc.

## About

## Pingod Addon Plugins

- Machine
- PinGodGame
- Resources
- AudioManager
- MemoryMap (optional, simulator)
