# MoonStation - PinGod
---

A simple rules table with novelty video mode using the [pingod-addons](https://github.com/FlippingFlips/pingod-addons)

### Features

- 2 Drop Target banks
- 2 Bumpers
- 2 Songs created specifically for this game (selected in menu)
- Bonus & Multipliers
- Video mode
