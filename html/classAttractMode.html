Detailed Description

A P-ROC (PinGodProcMode) but reusing the default PinGod Attract.tscn.

When the mode starts the ATTRACT_SCENE is loaded into the tree.

This mode handles start button to remove this mode and start the game if the P-ROC trough is full.
